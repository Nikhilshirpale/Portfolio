**Deployment Instructions (Git not installed yet):**
1. Download Git: https://git-scm.com/download/win
2. Install, restart terminal/VSCode
3. `git init`
4. `git add .`
5. `git commit -m "Initial portfolio"`
6. Create new repo on GitHub.com (name: portfolio)
7. `git remote add origin https://github.com/YOUR_USERNAME/portfolio.git`
8. `git branch -M main`
9. `git push -u origin main`
10. GitHub repo Settings > Pages > Branch: main, folder: /(root)
11. Site live at https://YOUR_USERNAME.github.io/portfolio
